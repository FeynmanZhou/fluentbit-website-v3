About

Fluent Bit

Fluent Bit is an open source and multi-platform data-log forwarder which aims to be a generic Swiss knife for events collection on any environment.

We, Treasure Data, as a Cloud Analytics company, provide an analytics infrastructure in the Cloud where we provide an end-to-end solution to collect, store and do analytics over the data. Fluent Bit is an integral part of this pipeline where it solve the log-forwarding problem for Cloud Native environments.

Flexible and Extensible Architecture

Since data collection is hard, a flexible and extensible architecture is always desired. The source of data can be many and it's likely multiple-formats are expected.
